To Setup Node Js server for Backend run the following command.

1) To install all the dependency:
```
cd backend
npm install
```

2) Start NodeJs server.
```
node server.js
```

http://localhost:8081/

To Setup React JS for Front end.

1) To install all the dependency:
```
cd ..
npm install
```

2) start ReactJs
```
npm start
```

http://localhost:3000/
